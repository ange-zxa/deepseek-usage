# DeepSeek Balance

> ⚠️ This is an **unofficial** third-party tool. Not affiliated with or endorsed by DeepSeek (深度求索).

A cute, minimal dashboard to monitor your DeepSeek API balance right in VS Code.

## Features

- Beautiful dashboard with real-time balance display
- Status bar indicator showing balance at a glance
- API key stored securely in system keychain (macOS Keychain / Windows Credential Manager)
- Auto-refresh every 30 seconds

## Setup

1. Click the ¥ icon in the activity bar
2. Click "Set API Key Securely"
3. Paste your DeepSeek API key (starts with `sk-`)
4. Your balance appears instantly

Your key is encrypted and stored in your OS keychain — never in plain text.
